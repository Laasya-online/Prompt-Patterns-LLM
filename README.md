# Prompt-Patterns-LLM

## Project Overview
This mini project explores how different prompt patterns change the behavior of a Large Language Model (Llama-3-8B-Instruct) using the Hugging Face Inference API in Google Colab.  
I used one article as input and applied three prompt patterns:
1. Simple summarizer
2. Executive tone rewriter
3. Role + constraints + example-based prompt

## Tools Used
- Google Colab
- Hugging Face Hub (`InferenceClient`)
- Model: `meta-llama/Meta-Llama-3-8B-Instruct`
